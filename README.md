# @charliesu/workflow-core

Shared workflow engine for TopFlow and TaraFlow.

## Installation

This is a private package. Install via npm link or file path:

```bash
npm link @charliesu/workflow-core
```

## Development

```bash
# Build
pnpm build

# Watch mode
pnpm build:watch

# Test
pnpm test
```
